Build a blog using hexo

Posted by PengboHu on 2017-02-09

hexo是一款基于Node.js的静态博客框架,我们可以将其搭建在github page上或者其他服务器上，但是由于github page速度慢而且空间少，所以我选择将其搭建在BAE上。下面我将介绍一下我如何将博客搭建在BAE上。

准备工作

  • 安装git
  • 下载node.js，下载通用版就可以

开始搭建博客

建立一个博客文件夹,并安装hexo

然后初始化一个博客
hexo init
这时一个简单的博客就已经完成，接下来看一下这个博客的样子,输入一下代码然后打开浏览器输入localhost:4000/就可以看到博客的外观。

这样一个最简单的博客就已经完成，接下来我们对博客进行优化。

博客优化

博客优化其实就是添加功能，网上有很多主题源码可以直接拿来修改。当然也可以自己制作主题，对于新手来说可以直接选择主题来用。可以在这里挑选喜欢的主题。

一般相应的主题仓库中都会有详细的说明教你怎么修改。下面我就简单的说一下大致流程，首先找到喜欢的主题之后，将代码clone到你的themes文件夹中，进入git bash,然后进入themes文件夹中。

然后更改根目录(注意不是主题根目录而是外面blog文件夹下的根目录)中_config.yml的themes字段
themes:choiced-themes
然后生成一下就可以看到简单的效果

如果成功就可以开始进一步修改_config.yml添加功能，至于怎么添加，相应的github仓库中都有详细的说明，就跟着说明文档改就可以了。修改过程不是很复杂但是很容易遇到各种问题，这个时候就要多看看github中的Issues了，当你把问题都解决的时候就会对hexo有一个深入的了解了。

至此一个博客就算完成了。接下来讲一下怎么部署到BAE中。当然也可以部署到github page上，部署到github page上的教程网上非常多，这里就不提了。
